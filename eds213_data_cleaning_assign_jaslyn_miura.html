Data cleaning

  1. Replacing the “.”, “-”, “n/a”, “unk” values with NA values.
# Replace non-numeric values with NA values.
snowcover_fixed <- snowcover_csv %>% 
  mutate(Land_cover = ifelse(Land_cover %in% c(".", "-", "n/a", "unk"), NA, Land_cover)) %>% 
  mutate(Water_cover = ifelse(Water_cover %in% c(".", "-", "n/a", "unk"), NA, Water_cover)) %>% 
  mutate(Total_cover = ifelse(Total_cover %in% c(".", "-", "n/a", "unk", "<row r=\\64\\ spans=\\1:11\\ x14ac:dyDescent=\\0.2\\>"), NA, Total_cover))
  1. We’ll now change the data types of the values within the Land_cover, Water_cover, and Total_cover to numeric/integer data types. This will allow us to use the Land_cover and Water_cover columns to properly calculate the Total_cover column.
# Convert land, water, and total values to `dbl` or numeric data type.
snowcover_fixed <- snowcover_fixed %>% 
  mutate(Land_cover = as.numeric(Land_cover)) %>% 
  mutate(Water_cover = as.numeric(Water_cover)) %>% 
  mutate(Total_cover = as.numeric(Total_cover))

This tells us that there may be values within our columns that are outside our range 0-100, representing percentages. We’ll fix this by filtering our data to exclude any rows where the values in the columns are outside of our range.

# Remove any rows where values are not a realistic percentage value.
snowcover_fixed <- snowcover_fixed %>% 
  filter(!(Land_cover < 0 | Land_cover > 100 |
  Water_cover < 0 | Water_cover > 100 |
  Snow_cover < 0 | Snow_cover > 100))
